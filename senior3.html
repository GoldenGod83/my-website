SkillsUSA Technical Computer Applications

Senior Year Project #3

In my senior year, I competed in the Connecticut State SkillsUSA Technical Computer Applications competition. I placed 2nd overall, representing Vinal Technical High School against the top IST students from 16 schools across the state.

  • Created and configured a Linux VM for cross-platform work.
  • Demonstrated technical skills in Microsoft Access, Excel, Word, and PowerPoint.
  • Completed tasks in networking.
  • Installed and configured OS environments (Windows/Linux), managed user accounts, and resolved troubleshooting scenarios.
  • Manipulated data using SQL and DML, handled data imports/exports, and built functional databases.
  • Built multi-layered Excel spreadsheets with functions, logic, and custom formatting.
  • Applied best practices in document formatting, slide design, and data analysis.

This competition challenged my full IT skillset across software, scripting, troubleshooting, and problem-solving under pressure.
