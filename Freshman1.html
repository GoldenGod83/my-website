Virtual Machines

Freshman Year Project #1

During my freshman year, I used Hyper-V to create live virtual machines. This allowed me to test a variety of operating systems like Windows and Linux without needing extra physical hardware.

  • Used ISO files to install different operating systems in virtual machines.
  • Practiced system setup and file sharing between host and VM.
  • Tested software in isolated environments without affecting the main PC.

This helped me learn how virtual machines work and gave me confidence in testing and troubleshooting new systems safely.
