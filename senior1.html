FREB - Franco's Real Estate Bot

Senior Year Project #1

For my senior project, I created a fully automated CRM system named FREB (Franco's Real Estate Bot) to handle lead generation, message creation, and communication for real estate prospects.

  • Used Textbelt API for outbound text message communication and real-time replies via a locally hosted webhook.
  • Developed a Python-based lead scraper using Selenium to extract live leads from the iSpeedToLead website.
  • Built a custom script to parse user intent and generate unique, human-like messages tailored to each lead’s real estate interests.
  • Implemented an approval flow so that users can review, edit, and approve AI-generated messages before sending.
  • Webhook server logs and routes replies directly into the system for easy follow-up and continued conversation threads.
  • All code is designed to run locally and cost-effectively, avoiding third-party CRM platforms.
