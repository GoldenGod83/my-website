FWB - Franco's Windows Breaker

Senior Year Project #4

FWB (Franco's Windows Breaker) is a custom Linux-based script I developed to automate bypassing the Windows login screen using the sticky keys backdoor method. This utility targets systems running Windows 7 through Windows 11.

  • Designed a winmod.sh script that replaces the sethc.exe binary with cmd.exe on a Windows partition.
  • This allows the user to press Shift five times at the lock screen to launch a full administrative command prompt.
  • Tested successfully on multiple Windows environments, verifying access and functionality across versions.
  • Learned advanced file system mounting, scripting, and Linux systemd service automation in the process.

FWB was developed as an academic proof of concept to better understand system security flaws and ethical hacking techniques.
